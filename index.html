Funciones de Input





































TIPOS DE INPUTS UTILIZADOS

Text: Permite ingresar texto corto en una sola línea, como nombres, usuarios o mensajes.

Number: Permite ingresar únicamente valores numéricos y puede incluir controles para aumentar o disminuir el número.

Date: Permite seleccionar una fecha mediante un calendario integrado.

Time: Permite seleccionar una hora específica, incluyendo horas y minutos.

Email: Campo diseñado para ingresar correos electrónicos, validando automáticamente su formato.

Datetime-local: Permite seleccionar una fecha y hora en conjunto, sin incluir zona horaria.

Password: Oculta los caracteres ingresados para proteger información sensible como contraseñas.

File: Permite al usuario seleccionar y subir archivos desde su dispositivo.

Tel: Campo para ingresar números telefónicos, optimizado para dispositivos móviles.

Checkbox: Permite seleccionar una o varias opciones de un conjunto.

Radio: Permite seleccionar solo una opción dentro de un grupo.

Submit: Botón que envía los datos del formulario.

Reset: Botón que limpia todos los campos del formulario.
